# Analista_Sist-mico-Estrat-gico
Motor de análisis estratégico construido en Rust. Fusiona Pensamiento Sistémico (Meadows) y Psicología del Poder (Urbina) para evaluar perfiles y entornos complejos mediante AI. Soporte nativo para PDF/DOCX, diseño Mobile-First y generación de informes ejecutivos A4.
