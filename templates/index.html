Operational Intelligence Guide

1. Tool Usage

This system is not a "chat". It is a structuring engine. To get useful results:

  • Context: Don't say "there are problems". Say "Budget was cut by 20% and the team fears layoffs".
  • Observations: Use behavioral facts ("Yelled at the meeting", "Goes silent when boss enters"). Avoid judgments ("He is toxic").
  • Audio: You can record meetings (with permission) or dictate unstructured notes. The system will structure them.

2. Scientific Basis (Human Motors)

Analysis is based on Self-Determination Theory (SDT) and Organizational Justice:

  • Autonomy: Need to control one's own life.
  • Competence (Mastery): Need to handle tasks effectively.
  • Relatedness (Belonging): Need to interact/connect with others.
